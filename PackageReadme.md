# mcpserver.azuredevops
MCP server as dotnet tool to access Azure DevOps.

## Installation
``` cmd
dotnet tool install --global mcpserver.azuredevops.stdio
```

<br>

## Supported Methods

### Project
- `GetProject`
- `GetProjects`


### Git
- `GetCommit`
- `GetCommitsForRepository`
- `GetRepository`
- `GetRepositories`
